subside /səb'said/

* nội động từ

- rút xuống, rút bớt (nước lụt...)

- lún xuống (nền nhà)

- ngớt, giảm, bớt, lắng đi

=storm subsides+ cơn bão ngớt

=tumult subsides+ sự ồn ào lắng dần

- chìm xuống (tàu)

- lắng xuống, đóng cặn

-(đùa cợt) ngồi, nằm

=to subside into an armchair+ ngồi sụp xuống ghế bành
