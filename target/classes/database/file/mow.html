- đống, đụn (rơm, rạ, cỏ khô); cót, cây (thóc)

- nơi để rơm rạ

* danh từ

- bĩu môi, nhăn mặt

- (nghĩa bóng) hạ xuống như rạ, tàn sát, làm chết như rạ (bệnh dịch...)
